How it works
  • 1
    Enter your details — paste the IP address from your VPS provider's dashboard and your Windows username.
  • 2
    Download the .rdp file — this file is generated entirely in your browser. Nothing is sent to any server.
  • 3
    Double-click to connect — Windows Remote Desktop Connection opens automatically. Enter your password when prompted.
  • 4
    Save it for next time — keep the .rdp file on your desktop for one-click access to your VPS.
Helpful Tips

Use a Strong Password

Set a long, complex Windows password on your VPS. RDP is exposed to the internet — brute-force attacks are common.

Change the Default Port

Consider changing RDP from port 3389 to a non-standard port in Windows Firewall to reduce automated attacks.

Allow RDP in Firewall

Make sure port 3389 (or your custom port) is allowed in both your VPS provider's firewall and Windows Firewall.

macOS & Linux

On macOS install Microsoft Remote Desktop from the App Store. On Linux use Remmina or FreeRDP.
